Phase 2: Implementation

weeks 10-17 · ~70 hours · build real understanding through code

This phase transforms conceptual knowledge into embodied understanding. By implementing neural networks from scratch, you'll develop intuitions that no amount of reading can provide.

the golden rule

Type everything yourself. Never copy-paste code. When you hit errors, celebrate them — debugging stories become interview gold.

week 10-11: micrograd

Video: Karpathy's "Building micrograd" (2h 25m)

goals

~15 hrs
  • Create GitHub repo: micrograd-from-scratch
  • Implement the Value class with autograd
  • Implement Neuron, Layer, MLP classes
  • Document all debugging struggles
