Phase 0: Foundations

weeks 1-3 · ~45 hours · Python + PyTorch fluency

Before diving into safety concepts, you need the technical foundation to engage with code, papers, and implementations. This phase builds Python fluency and PyTorch basics — the lingua franca of modern ML.

goal

Be able to write a basic neural network in PyTorch, understand what .backward() does, and have solid Python fundamentals. You don't need to be an expert — just fluent enough to follow Karpathy's videos in Phase 2.

week 1: python fundamentals

Resource: LearnPython.org (free, interactive)

Learn the Basics (13 lessons)
  1. Hello, World!
  2. Variables and Types
  3. Lists
  4. Basic Operators
  5. String Formatting
  6. Basic String Operations
  7. Conditions
  8. Loops
  9. Functions
  10. Classes and Objects
  11. Dictionaries
  12. Modules and Packages
  13. Input and Output

week 1 goals

~15 hrs
  • Complete all 13 "Learn the Basics" lessons
  • Create 20-25 Python Anki cards
  • Fill 5+ pages of analog notes
  • Write syntax patterns by hand
certification available

After completing the tutorials, you can get certified at LearnX and add the certification to your LinkedIn profile.

week 2: python for data

Continue with LearnPython.org advanced sections + add NumPy basics.

week 2 goals

~15 hrs
  • Complete "Data Science Tutorials" section on LearnPython.org
  • NumPy basics: arrays, shapes, broadcasting
  • Practice: Write 3-5 small scripts from scratch
  • Create 15 more Anki cards
key concepts to internalize
  • List comprehensions
  • Dictionary methods (.get(), .items())
  • Class __init__ and self
  • NumPy array shapes and broadcasting
  • Common debugging patterns

week 3: pytorch fundamentals

Resource: Daniel Bourke "PyTorch in a Day" (~10 hours)

week 3 goals

~15 hrs
  • Watch first 5 hours, coding along
  • Understand tensors, operations, autograd
  • Build your first neural network
  • Understand the training loop
  • Create 20 PyTorch Anki cards
pytorch concepts to internalize
  • torch.tensor vs NumPy arrays
  • Tensor shapes: .shape, .view(), .reshape()
  • Device management: CPU vs GPU
  • requires_grad and .backward()
  • nn.Module and forward()
  • The training loop: forward, loss, backward, step
the golden rule

Type everything yourself. Never copy-paste code. The physical act of typing builds neural pathways that passive watching cannot. When you hit errors, celebrate them — debugging stories become interview gold.

phase 0 completion checklist

  • Can write Python classes from scratch
  • Understand list comprehensions, generators
  • Can manipulate NumPy arrays confidently
  • Can create tensors, move to GPU
  • Can build and train a simple neural network
  • Have ~55-60 Anki cards total
  • Have 15+ pages of analog notes
